# Interview Guide

### The Pitch
"EduQuiz Architect is an AI-powered platform that helps educators turn textbooks into interactive quizzes. It uses Gemini 2.5 to read documents and creates personalized learning paths for students based on real-time performance."

### Technical Highlights
- **GenAI Pipeline**: Built structured flows with Genkit for robust text analysis and question generation.
- **Real-time State**: Used Firestore for low-latency updates to student cognitive profiles.
- **Adaptive Logic**: Implemented an engine that adjusts question difficulty dynamically.
- **Clean Architecture**: Leveraged Next.js 15 Server Components and non-blocking Firestore mutations.

### Key Skills Demonstrated
- Full-stack development with Next.js 15.
- AI Integration (LLMs, Structured Outputs).
- Real-time NoSQL database management.
- Modern UI/UX with ShadCN and Tailwind.
